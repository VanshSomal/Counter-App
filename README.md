# Simple Counter App

A clean, minimalist, and responsive counter application built to demonstrate core concepts of state management and UI design in **[Insert Language JavaScript]**.

## 🚀 Features

* **Increment & Decrement:** Easily add or subtract from the current count.
* **Reset functionality:** Clear the counter back to zero with a single click.
* **Responsive Design:** Works flawlessly on mobile, tablet, and desktop screens.
* **Persistent State:** Saves the counter value even after refreshing the page (via LocalStorage/SharedPreferences).

## 🛠️ Tech Stack

* **Frontend:** HTML, Javascript
* **Styling:** Tailwind CSS
